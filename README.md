# automation-practice-selenide
Projeto de teste utilizando Selenide e cucumber

## Ferramentas usadas para a criação 
* Selenide
* Selenium
* Cucumber
* TestNg
* Junit

## Pre requisitos
Para execução dos testes são necessario ter

* Instalado Git
* Instalado IntelliJ
* Instalado JDK
* Instalado Maven
* Setup Project
